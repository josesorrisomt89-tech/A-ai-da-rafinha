Guia de Publicação

Tutorial: Como Atualizar o Cardápio Para Seus Clientes

O processo é simples e consiste em três etapas principais:

  1. Fazer suas alterações no painel de administração.
  2. Exportar um arquivo com essas novas informações.
  3. Substituir o arquivo antigo no código do seu projeto e publicar.

Passo 1: Faça as Alterações no Painel Admin

Tudo começa no seu painel de gerenciamento. É aqui que você adiciona produtos, muda preços e ajusta as configurações.

  1. Acesse o Painel Admin do seu sistema (clicando em "Entrar" na tela do cardápio e digitando seu PIN de administrador).
  2. Navegue pelas seções que deseja alterar:
    • Para adicionar ou editar açaís, sorvetes, bebidas, etc., vá para "Produtos".
    • Para alterar preços ou nomes de adicionais, vá para "Adicionais".
    • Para mudar as categorias (Açaí, Bebidas, etc.), vá para "Categorias".
    • Para ajustar informações da loja, horários ou taxas de entrega, vá para "Configurações".
  3. Faça todas as modificações que desejar. Elas são salvas automaticamente no armazenamento local do seu navegador.

Passo 2: Exporte o Arquivo de Dados Atualizado

Depois de fazer todas as alterações, você precisa gerar o arquivo que será usado para atualizar o cardápio público.

  1. No menu lateral do Painel Admin, clique na seção "Configurações" (o ícone de engrenagem ⚙️).
  2. Dentro da tela de "Configurações", clique na aba "Operacional".
  3. Na seção "Como Publicar Suas Alterações Para Clientes", você encontrará as instruções e um botão.
  4. Clique no botão azul "Exportar Dados para Publicação".

Ao clicar, seu navegador fará o download de um arquivo chamado menu-data.ts. Este arquivo contém todo o seu cardápio e configurações atualizadas.

Passo 3: Substitua o Arquivo Antigo no Projeto (Parte Crítica)

Agora, você precisa pegar o arquivo que baixou e colocá-lo no lugar do arquivo antigo na estrutura de pastas do seu projeto. Esta é a parte mais importante.

  1. Abra a pasta onde está o código do seu projeto.
  2. Navegue até o seguinte caminho: src/data/.
  3. Dentro da pasta data, você encontrará um arquivo já existente chamado menu-data.ts.
  4. Exclua ou apague este arquivo antigo.
  5. Agora, pegue o novo arquivo menu-data.ts que você baixou no Passo 2 e mova-o ou copie-o para dentro desta mesma pasta src/data/.

Resumindo: O arquivo menu-data.ts que você acabou de baixar deve substituir o que estava na pasta src/data/.

Passo 4: Publique as Alterações na Vercel

Com o arquivo novo no lugar, o último passo é publicar a nova versão do seu site para que as alterações fiquem online.

  1. Salve todas as alterações nos arquivos do seu projeto.
  2. Envie as suas alterações (faça o "commit" e "push" via Git) para o serviço onde seu site está hospedado (Vercel, GitHub, etc.).
  3. O seu serviço de hospedagem irá detectar as novas alterações e automaticamente criar uma nova versão ("build" e "deploy") do seu site com o cardápio atualizado.

Após a publicação ser concluída, todos os clientes que acessarem seu site verão imediatamente as novas alterações, sem problemas de cache!
